Список задач:               npm run reddir
Создать задачу:             npm run add "Название задачи" "Текст задачи"
Открыть задачу:             npm run open "Название задачи"
Переименовать задачу:       npm run rename "Старое название" "Новое название"
Отредактировать задачу:     npm run update "Названи задачи" "Новое описание задачи"
                            (ВНИМАНИЕ: Тектс задачи заменяется полностью)
Удалить задачу:             npm run del "Название задачи"
Удалить все задачи:         npm run delall
